# Interactive Story Game

A simple text-based interactive story game built with React. Make choices, follow the story, and explore different outcomes!

## Features

- Text-based narrative displayed one scene at a time
- Multiple choices for each scene
- Progress saved in localStorage
- Smooth transitions between scenes

## How to Play

1. Open the app in your browser.
2. Start a new game or continue a saved game.
3. Read the story and select your choices.
4. Your progress is saved automatically.

## Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
    ```

2. Install Dependencies
    ```bash
    npm install
    ```

3. Run the app
    ```bash
    npm run dev
    ```

##Technologies Used
- React
- Tailwind CSS
